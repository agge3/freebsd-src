# System call creation library
## Usage
All files will be generated by main.lua<br>
Files are associated by their module, and modules can be called as standalone script to generate that specific file
## Directories
* root
 * <code>main.lua</code>, main entry, calls all the modules
 * <code>config.lua</code>, contains the global config table and associated config functions
 * core (core classes)
  * <code>syscall.lua</code>, each system call entry in syscalls.master is packaged into a system call object
  * <code>scarg.lua</code>, each argument for the system call is packaged into an argument object
  * <code>scret.lua</code>, an object for the return value of the system call
  * <code>freebsd-syscall.lua</code>, the master system call table (after processing)
 * modules
  * <code>syscalls.lua</code>, generates syscalls.c
  * <code>syscall_h.lua</code>, generates syscall.h
  * <code>syscall_mk.lua</code>, generates syscall.mk
  * <code>init_sysent.lua</code>, generates init_sysent.c
  * <code>systrace_args.c</code>, generates systrace_args.c
  * <code>sysproto_h.lua</code>, generates sysproto.h
* tools
  * util.lua, utility functions
  * bsdio.lua, I/O for the library
